🛡️ 安全问题修复（附录 D 全部解决）
严重CLI 转义/引号处理 → CliInterpreter 完整分词引擎
严重日志 API Key 泄露 → Sanitizer 5 种 Key 脱敏
严重ReAct 格式敏感 → PromptEngine 模糊匹配+中英文冒号
严重HTTP 连接池 → RemoteApi Ktor 连接池
严重后台进程被杀 → ShellService 前台服务
严重磁盘写满 → StorageMonitor 预检+自动清理
严重无限循环 → PromptEngine.detectLoop 命令循环检测
严重日志脱敏不完整 → Sanitizer 链式多模式处理
中等命令注册空指针 → CommandRegistry 空安全
中等命令历史溢出 → 自动缩容
中等截图文件冲突 → ScreenshotManager UUID+时间戳
中等模型路径无校验 → LocalModel 初始化校验
中等回调内存泄漏 → ViewModel 生命周期管理
中等网络切换失败 → Ktor 超时+重试
中等扩展版本兼容 → ExtensionLoader 语义版本检查
中等fs.write 路径穿越 → SecurityPolicy 危险模式检测
